# Ovumcy Brand Assets

Files:
- `ovumcy-icon.svg`: square app/icon mark for light backgrounds
- `ovumcy-icon-dark.svg`: square app/icon mark for dark backgrounds
- `ovumcy-logo-horizontal.svg`: lockup (symbol + wordmark) for light backgrounds
- `ovumcy-logo-horizontal-dark.svg`: lockup for dark backgrounds

Core palette:
- Lavender: `#A989E7`
- Coral accent: `#FF7F93`
- Wordmark (light bg): `#4A3D6A`
- Wordmark (dark bg): `#F6F3FF`

Usage notes:
- Keep clear space around the symbol at least `0.5x` of icon width.
- For 32x32 contexts (favicon/app icon), use `ovumcy-icon.svg`.
- Do not add shadows, heavy gradients, or extra details.
